Assignment Instructions

Download Reference Image

  1. Create a geodatabase and define the WGS 1984 Web Mercator Zone 36 coordinate system.
  2. Create the following layers:
    • Agricultural
    • Urban Land
    • Streets
    • Landmarks
    • Vacant
    • Water Area
  3. Define attributes for each layer as follows:
    • Agricultural: Street name, Governorate name, Type (Park, Garden, Natural Reserve, Agricultural Land), Tax value (0-10,000).
    • Urban Land: Zone ID, Population Density (Low/Medium/High), Zone Type (Residential/Mixed-use/Rural), Street name, Governorate name, Price (0-1,000,000), Number of floors.
    • Streets: Road Type (Highway/Main/Secondary/Local), Length (Meters), Road ID, Street name, Governorate name.
    • Landmarks: Landmark ID, Type (Pharmacy, Mall, Mosque, etc.), Place name, Street name.
    • Vacant: Type, Governorate Name (Alexandria).
  4. Draw the phenomena in the study area and add dummy data to each layer. Activate tracking for all layers.
  5. Set symbology:
    • Urban Land: Use unique values based on building type.
    • Landmarks: Assign symbols from a predefined gallery.
  6. Topology Requirements:

    Create a Topology inside the feature dataset and ensure spatial data integrity using the following rules:

    • Urban Land:
      • Must Not Overlap
      • Must Not Have Gaps
      • Must Be Covered By Boundary of Water Area (if applicable)
    • Agricultural:
      • Must Not Overlap
      • Must Not Intersect Urban Land
      • Must Be Covered By Boundary of Streets
    • Streets:
      • Must Not Overlap
      • Must Not Have Dangles (except permitted dead-ends)
      • Must Not Self-Intersect
    • Vacant:
      • Must Not Overlap
      • Must Not Intersect Urban Land
    • Water Area:
      • Must Not Overlap
      • Must Cover All Internal Islands Completely

    Validate the topology, fix errors, and include a screenshot of the topology error inspector in the final deliverables.

  7. Create a professional map layout that includes: North Arrow, Coordinate System, Data Source, Scale Bar, Legend, Layout Date and Data Date, Author, Logo, Locator Map, and a Table listing top five regions by building price.
  8. Export outputs: Map (JPG/PDF), Streets layer (KML), Buildings layer (Excel). Organize deliverables in a single folder.

Expected Outputs

  • JPG and PDF files for the map layout.
  • KML file for the Streets layer.
  • Excel file for the Buildings layer.
  • GDB and APRX project files.
